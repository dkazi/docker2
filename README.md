
											AI Log Analysis API

Ένα ολοκληρωμένο σύστημα ανάλυσης αρχείων καταγραφής (logs) ασφαλείας που χρησιμοποιεί Τεχνητή Νοημοσύνη για τον εντοπισμό επιθέσεων και την παροχή συμβουλών σε πραγματικό χρόνο.

Αρχιτεκτονική & Λογική
Το σύστημα αποτελείται από τρία κύρια μέρη που λειτουργούν σειριακά:

1. Εκπαίδευση & Μοντελοποίηση (GOD_OF_AI_TRAINING.py)
Σκοπός: Δημιουργεί τη βάση γνώσης για τις επιθέσεις.

Λογική:

Χρησιμοποιεί τη βιβλιοθήκη drain3 για να ομαδοποιήσει εκατομμύρια logs σε συγκεκριμένα "πρότυπα" (templates).

Εφαρμόζει μια Ταξινομία Επιθέσεων (Attack Taxonomy) που αντιστοιχίζει μοτίβα (Regex) σε κατηγορίες κινδύνου (π.χ. SQL Injection, Brute Force) και τεχνικές MITRE.

Αποθηκεύει τα αποτελέσματα σε μια διανυσματική βάση δεδομένων (ChromaDB) για γρήγορη αναζήτηση.

2. Εντοπισμός & Ανάλυση (GOD_OF_DETECTION.py)
Σκοπός: Επεξεργάζεται τα logs και εντοπίζει ύποπτες δραστηριότητες.

Λογική:

Διαβάζει τα αρχεία καταγραφής και τα συγκρίνει με τα εκπαιδευμένα πρότυπα.

Υπολογίζει ένα Threat Score (0-100) για κάθε IP, βασιζόμενο στη σοβαρότητα και τη συχνότητα των επιθέσεων.

Εξάγει τα αποτελέσματα σε ένα αρχείο detection_results.json, το οποίο περιλαμβάνει στατιστικά, αλυσίδες επιθέσεων (attack chains) και δείγματα logs.

3. Διεπαφή Χρήστη & AI Chat (GOD_OF_CHAT.py)
Σκοπός: Επιτρέπει στον αναλυτή SOC να κάνει ερωτήσεις σε φυσική γλώσσα.

Λογική:

Intent Detection: Αναγνωρίζει τι ζητάει ο χρήστης (π.χ. "τι έγινε τις τελευταίες 4 ώρες;", "πες μου για την IP X").

Context Building: Δημιουργεί ένα στοχευμένο κείμενο (context) με τα σχετικά δεδομένα από το detection_results.json.

OpenAI Integration: Στέλνει το context στο GPT-4, το οποίο απαντά ως έμπειρος αναλυτής ασφαλείας, προτείνοντας συγκεκριμένους κανόνες firewall ή ενέργειες.

										AI Log Analysis GUI

Είναι ένα web app το οποίο αναλύει logs σε πραγματικό χρόνο που χρησιμοποιεί Generative AI (OpenAI) για τον εντοπισμό απειλών, την οπτικοποίηση δεδομένων ασφαλείας και την παροχή συμβουλών από έναν "AI SOC Analyst".

Δυνατότητες
Real-time Monitoring: Παρακολούθηση πολλαπλών αρχείων log ταυτόχρονα (π.χ. auth.log, syslog).

SOC Analysis Tab: Εξειδικευμένο chatbot που τρέχει incremental detection runs και απαντά σε ερωτήσεις βάσει των ευρημάτων.

Interactive Dashboard: Οπτικοποίηση δεδομένων με charts (Activity by hour, Top IPs, Event breakdown).

Live Log Stream: Προβολή των logs καθώς δημιουργούνται με δυνατότητα φιλτραρίσματος.

Incident Export: Εξαγωγή των συνομιλιών με την AI σε αρχεία PDF για αναφορές περιστατικών.

IP Flagging: Σύστημα σήμανσης (flagging) ύποπτων IP διευθύνσεων με δυνατότητα προσθήκης σημειώσεων.
